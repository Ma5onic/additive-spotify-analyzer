This site is a collection of various types of analysis on your Spotify data. Top artists shows a text graph of your artists and genres as known by Spotify. Tracks list is a track list ordered by last added. Current work involves Plotly and data clustering using scikit-learn packages to plot features against time to analyze evolution (or lack thereof) of musical tastes. Orphaned tracks analysis is trying to find all tracks that have been abandoned. This means that you never added any albums of this artist. Maybe you liked one song and forgot about this cool band? Or maybe it's a stinker?
